----------/o que permite que o java se conecte com qualquer despositivo/--------
																			
	O que torna possivel ao java rodar em qualque sisitema operacional é a  
(JVM ou "java virtual machine"), o prcesso funciona da seguinte forma voce cria 
o codigo em java e compila usando o compilador e esse codigo é tranformado no (b-
ite code) a maquna virtual que é istalada no sistema interpreta este byte code 
executando a aplicação java.						      
	Para o java funsinar no seu computador ou em qualquer outro dispositi-
vo é necessario ter esta maquina virtual intalada.			      
	Por exemplo a gente custuma installar o java na sna verdae voce esta i-
nstalando  o JVM no seu computador quando voce vai executar o programa da rece-
ita federal(por que esse programa da receita federal é feito em java).
